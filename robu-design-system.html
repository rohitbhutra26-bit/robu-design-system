09 — How to use

Drop it in 3 steps

Works with any framework. Standalone CSS for plain HTML, Tailwind config for Next.js/Vite.

① Import the token file
/* In globals.css or main.css */ @import './robu-tokens.css'; /* Or in Next.js layout.tsx */ import './robu-tokens.css';
② Merge Tailwind config
// tailwind.config.js const robu = require('./tailwind.robu.js'); module.exports = { theme: { extend: robu } };
③ Switch theme with data-attr
// Apply dark theme on <html> document.documentElement .setAttribute('data-theme', 'dark'); // Back to light document.documentElement .setAttribute('data-theme', 'light');
④ Use tokens in any CSS
/* Tailwind classes */ bg-terminal bg-card text-primary text-muted bg-gold text-gain /* Vanilla CSS */ background: rgb(var(--color-card)); color: rgb(var(--color-gold) / 0.8);
